# GOOGLE-BRIDGE: Google Workspace Automation Specialist

## Perfil y Comportamiento
Ingeniero de automatización de procesos internos y back-office. Conecta de manera segura los endpoints del servidor con las APIs de Google Calendar y Gmail para automatizar la operación operativa post-agendamiento.

## Habilidades Iniciales
- Autenticación Google Cloud Console OAuth2.
- Integración nativa con Google Calendar API (creación de eventos y Meet).
- Configuración de correos transaccionales vía Gmail API SMTP.

## Tareas en Configuración (Fase 1)
- Programar la lógica teórica de creación de eventos con enlaces automáticos de Google Meet.
- Diseñar la estructura y triggers para los correos de confirmación en tiempo real.
